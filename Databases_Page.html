MySQL

Code

Meaning
mysql -u root -p Lauches MySQL as the root user in linux or Mac.
mysql -u root -p database_name < file_name.sql Saves a database as a .sql file.
mysql -u root -p database_name < my/path/file_name.sql Loads a .sql file as a database.
exit; Back to the terminal in standard mode.

Basic SQL

Code

Meaning
CREATE DATABASE databasename; Creates a database with a given name (name in lowercase).
USE databasename; Tells mySQL to use a given database.
SHOW DATABASES; Shows the existing databases.
CREATE TABLE; Creates a table.
CREATE TABLE user (
id INTEGER NOT NULL AUTO_INCREMENT PRIMARY KEY,
lastname VARCHAR(100),
firstname VARCHAR(100),
email VARCHAR(255) NOT NULL UNIQUE
);		 Creates a table named "user" with 4 columns.
PRIMARY KEY (option) Mandatory field for the id of each element.
INTEGER or INT (type) Integer numbers field.
NOT NULL (option) This field cannot be null or empty.
AUTO_INCREMENT Auto generated field.
VARCHAR(100) (type) Text field limited to 100 characters.
UNIQUE (option) The field cannot avec the same value twice.
CREATE TABLE aliment (
id INTEGER NOT NULL AUTO_INCREMENT PRIMARY KEY,
nom VARCHAR(100) NOT NULL,
marque VARCHAR(100),
sucre FLOAT,
calories INTEGER NOT NULL,
graisses FLOAT,
proteines FLOAT,
bio BOOLEAN DEFAULT false
);		 Creates another table with 8 columns.
FLOAT (type) The field will contain decimal numbers.
BOOLEAN (type) Field that can take only true or false.
DEFAULT (option) Declares a default value.
SHOW tables; Displays the existing tables.
FROM tablename Selects a table.
SHOW COLUMNS FROM tablename; Displays the columns of a given table with their types and options.
INSERT INTO `table_name` (`column1`, `column2`, `column3`)
VALUES ('value1', 'value2', 'value3'); 		Select a given table, targets the columns and adds the needed values.
INSERT INTO `table_name` (`column1`, `column2`, `column3`)
VALUES
('value1', 'value2', 'value3'),
('value1', 'value2', 'value3'),
('value1', 'value2', 'value3'); 		Inserts several objects in the database.
`tables`, `columns`, "text", "varchar", boolean, integer, float Backticks, double quotes, no ticks.
SELECT * FROM table_name; Displays everything from a given table.
SELECT * FROM table_name WHERE 'id' = '1'; Filters everything but the first row.
SELECT `column1`, `column2`, `column3` FROM table_name; Displays only the given columns.
UPDATE `user` SET `email` = 'quentind@gmail.com' WHERE `id` = '1'; Targets the table, then the column and finaly the row with 'id'.
UPDATE `user` SET `email` = 'quentind@gmail.com', `car_brand` = `cadillac` WHERE `id` = '1'; Changes two columns for the same row.
UPDATE `table_name` SET `column` = "value"; Updates all the elements in a given column.
DELETE FROM `user` WHERE `id` = '2'; Deketes the second row.
DELETE FROM `table_name`; Deletes everything in the table.
DROP TABLE `table_name`; Remove an entire table.
DROP DATABASE `table_name`; Remove an entire database.

Advanced SQL

Code

Meaning
SELECT * FROM table_name WHERE column_name = "value"; Displays any object containing a given value. Also works with UPDATE and DELETE.
SELECT * FROM table_name WHERE column_name < number; Works with any operator (<, >, <=, >=). Only workks with numbers.
SELECT * FROM table_name WHERE column_name LIKE "text"; Same command but with text.
"%end_of_text", "beginning_of_text%", "Ã¹middle_of_text%" Referes to any text that ends a given way, starts a given way and contains a given content.
SELECT * FROM table_name ORDER BY column_name ASC/DESC; Select a column and order the different values ascending or descending (works with numbers and text).
SELECT * FROM table_name WHERE column_name < number ORDER BY column_name DESC; Combines the filter and descending order.
SELECT *
FROM aliment
WHERE (calories < 90) AND (sucre >10)
ORDER BY calories DESC;		 Command on several lines, easier to read.
SELECT COUNT(*)
FROM table_name
WHERE column_name LIKE "%end_of_text";		 Returns the number of element which the column value ends a given way.
SELECT COUNT(column_name)
FROM table_name
WHERE column_name LIKE "%end_of_text";		 The same but only taking a column.
SELECT COUNT(DISTINCT column_name)
FROM table_name
WHERE column_name LIKE "%contains_text%";		 Counts how many items share a given text but are not the same (so we know there are no duplicates).
SELECT COUNT(DISTINCT column_name) AS "alias_name" FROM table_name WHERE column_name LIKE "%contains_text%"; Creates an alias.
AVG(), SUM(), MAX(), MIN()... Returns the average, the sum, the max value and the min value : functions.
SELECT MAX(column_name) AS "alias_name"
FROM table_name; 		Returns the max value of a given column inside a given table as an alias.
SELECT AVG(column_name) AS "alias_name"
FROM table_name
WHERE column_name > number;		 Returns the average value of the elements superior to a given number of given column inside a given table as an alias.
SELECT ROUND(AVG(column_name)) AS "alias_name"
FROM table_name
WHERE column_name > number;		 Same as before but rounds it at the superior integer.
