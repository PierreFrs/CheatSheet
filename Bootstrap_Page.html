Bootstrap Grid

Code

Meaning
class='container' Creates a container (centered with margin).
class='container-fluid' Creates a container a full width container.
class='row' Creates a row.
class='column' Creates a column.
class='col-(number of column width)' Used to set the width of a column (on a 12 bases).
class='col-auto' Fit the content.
class='col-(breakpoint)-(size)' Sets the size of a column (1-12) for a certain breakpoint (sm, md, lg, xl).
