JSX

Code

Meaning
const 'variable name' = 'HTML element' The way to declare an HTML variable in react.
(multiline elements) Use () if multiline HTML elements.
ReactDOM.render('variable', 'targeted element') How to actually display elements.
class vs className In JSX we use className='' instead of class=''.
<br /> Self closing tags must include a forward slash.
<HTML tag>{javascript expression}<HTML tag> You need to surround a regular JS code around {} for it to be read as plain JS inside HTML tags.
onClick = {}, onMouseOver = {}... JSX event listeners.
key='' Used with <li> for React to remember the order.
const myFunc = React.createElement() A way to create react elements without JSX

React Components

Code

Meaning
import React from 'react'; The way to import React library.
import ReactDOM from 'react-dom'; The way to import the ReactDOM
class YourComponentNameGoesHere extends React.Component {} The way to declare React a class component (must begin with a capital letter).
render() {} Render method syntax.
<MyComponentClass /> Syntax to declare a new component.
ReactDOM.render(
<MyComponentClass />,
document.getElementById('app')
);		 Syntax to render a component.
this. Refers to an object on which this's enclosing method.
<div onClick={this.myFunc}></div> Syntax to add an event handler.
import/export Imports and exports different components between JS files.
this.props Syntax to see a props object.
this.props.children Returns the child of a JSX element or its children in an array.
Example.defaultProps={} Sets a default props in the form of an object.
constructor(props) {
super(props);
this.state = { mood: 'decent' };
}		 Syntax to declare a state (here: mood: 'decent')
this.state.property Syntax to access a property of a state.
this.setState({updated object}) Update state.
this.methodName = this.methodName.bind(this) Syntax to define an event handler (bind !!!).
