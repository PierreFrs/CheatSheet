Basics

Code

Meaning
python3 (in bash) Opens python session in the terminal.
exit() Exits python in the terminal.
variableName = "string" Declares a variable.
word_word A good practice for declaring python variable (the snake case) is to separate each word in the variable by an underscore.
Start the variable by _ or a letter Can't start with a number.
Only use _, letters, numbers Don't use symboles or accents.
type(variableName) Gives the variable type (integer, float, string, boolean).
listName.append('value') Adds a value at the end of the list.
listName.remove('value') removes a value at the end of the list.
listName.pop(index) Removes a number at a given place.
len(variableName/listName) Displays the length of the variable, string, list...
listName.sort() Sorts alphabeticaly or numericaly a list.
