File System

Command Result
/ Root directory
Sub-directory Purpose
/bin common binary executables used by all users
/boot
/dev attached devices (usb, cdrom, mouse, keyboard)
/etc configuration files
/home personal directories for each user account
/lib shared system libraries
/media directory for mounting removable devices (floppy drive, cdrom)
/mnt directory for mounting filesystems (nfs, smb)
/opt optional vendor add-on software
/proc virtual filesystem for system processes/resources information
/root home directory for administrator account
/run storage for runtime information

/srv data for server services

/tmp temporary files purged on reboot
/usr utilities and read-only user data/programs
/var variable and log files

Bash

Command Result
pwd Print Working Directory (file path from root to current directory).
mkdir Make Directory (creates a new directory in the current directory or at the end of a given file path).
ls List (lists the content of directory).
Lists options :
  • -a : lists all content (i. hidden files and directories)
  • -l : lists content in long format
  • -t : orders files and directories by the time they were last modified.
  • -alt : combines the previous three.
  • ls directory/* lists the sub-directories and their content.
cd Change Directory (go to a given directory : full file path, name of directory's children or the parent (with ..))
touch Create New File (touch 'name of file').
clear Clears the terminal
chmod Changes the permission type (more here !
cat Displays the content of the file (cat fileName).
cp source.txt destination.txt Moves a file from a source to a destination directory.
cp source.txt source.bak Make a copy of the file with a new extension (bak = backup).
cp source.txt directory Creates a copy of the file in another directory.
cp file1.txt file2.txt directory/ Copies multiple files in the destination directory.
* Wildcard : cp * directory/ (copy all the file in the current directory to the destination directory), cp w*.txt directory/ (copy files that start with 'w' and ends with '.txt')
mv Moves (used like cp).
rm Removes files.
rm -r Remove directories.
echo 'text' Displays 'text' in the terminal.
echo "text" > file.txt Redirects content to a file (!!! overwrites).
wc Wordcount Command (counts the number of lines, words and characters in a file).
wc -l cont the number of lines.
cat file1.txt >> file2.txt Adds to a file without losing the original content.
cat > file.txt Redirects standard input into a command.
| Pipe : redirects the standard output of a command to another command.
sort file.txt Takes the standard input and orders it alphabetically.
uniq Filters out duplicates in a file. More here !
grep Global Regular Expression Print : searches files for lines that matches a pattern.
grep -i Enables the grep command to be case Insensitive.
grep -R Searches all files in a directory and outputs filenames and lines containing matched results.
grep -Rl Outputs only filenames with matched results.
sed Stream Editor : accepts standard inputs and modifies it based on an expression.
sed 's/'string'/'rstring/g' file.txt s : substitution, 'string' : the string to replace, 'rstring' : replacement string, g : global (replaces all targeted strings on the line).
sed -i Rewrites the file.
~/.bash_profile Accesses the bash profile.
nano ~/.bash_profile Opens bash profile in nano to edit it.
source ~/.bash_profile Activates the bash profile changes right away.
echo $'variable' Displays the variable
env Displays all the envirenement variables (preferences and settings).

Nano

Command Result
nano Opens nano. Documentation here !
Ctrl + X Quits nano
Ctrl + 0 Saves a File
Ctrl + G Opens help menu
echo "smthg" (in profile) Displays something at the start of the session.
Command aliases (in profile) alias shortcut="command"
export (in profile) Sets the used variable in all child sessions.
USER="" Variable that sets the computer's owner's name.
PS1 Defines the makeup and style of the command prompt.
HOME Variable that displays the path of the home directory ~.
PATH variable that stores a list of directories separated by a colon.
