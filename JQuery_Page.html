Effects

Code

Meaning
.hide() Hides the targeted element.
.show() Shows the targeted element.
.toggle() Acts as a switch (without any parameter it shows or hides an element).
.fadeIn()/.fadeOut() Make an element appear or disappear with a fade.
.fadeToggle() Switches between fade in and fade out.
.slideUp()/.slideDown()
.slideToggle() Switches between slide up and down.

Mouse Events

Code

Meaning
'click' Listens for a mouse click.
'mouseenter' Listens for the mouse to enter an element's area.
'mouseleave' Listens for the mouse to leave an element's area.
$('.example-class').on(event, callback).on(event, callback); Chains events.
$(event.currentTarget) Only selects the element on which an event occurs.
'keyup' Listens for a key release.
'change' Listens for a value change.

Style Methods

Code

Meaning
.css(property, value) Targets a css property and change its value.
.css({key/value pairs}) Stores multiple css properties/values inside a JS object.
.animate({value/key object}, animation time) Sets one or multiple properties to animate and the time it takes.
.addClass() Adds a class to an HTML element.
.removeClass() Remove a class from an HTML element.
.toggleClass() Switches a class on & off.
.val() Targets the value of an element.

Traversing the DOM

Code

Meaning
.children() Target all the children elements of a parent element.
.parent() Targets the parent element of an element.
.siblings() Targets the adjacent elements of an element.
.closest('element name') Targets the the closest element with the specified name up the DOM tree.
.next()/.prev() Only targets the next/previous sibling.
.find() Targets one or multiple elements that are descendants of a specified element.
